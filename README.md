# Mechanical Task 2

## 3D Heart Keychain

This project was designed using Onshape to create a 3D heart-shaped keychain.

### Design Process
- Created a new sketch on the Front Plane.
- Drew the heart shape using sketch tools.
- Adjusted the dimensions to achieve a smooth and symmetrical design.
- Added a circular hole for the keychain ring.
- Used the Extrude feature to convert the 2D sketch into a 3D model.
- Added the text "Aim Higher" on the front face.
- Modified the text position and size to fit the heart design.
- Reviewed the model and made final adjustments to improve the overall appearance.
- Exported the final model as an STL file.
